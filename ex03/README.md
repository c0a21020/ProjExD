# 第3回
## 空飛ぶこうかとん(ex03/maze.py)
＜使い方＞
実行すると、空を飛んでいるこうかとんが雲の道を進む迷路がででくる。キーボードの←↓↑→のキーを押すことで進むことができる。
### 追加した機能
- 四則演算:+以外の四則演算をする。
- オールクリア:entryに入力されている数字，数式の文字列全体をdeleteする。
- 入力文字を表示する枠の背景色をつけ、入力された文字列の色もつけた。
-ボタンに色をつけた。
